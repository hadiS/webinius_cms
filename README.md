# WebiniusCms
WebiniusCMS is a rails CMS engine for developers.

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'webinius_cms'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install webinius_cms
```
